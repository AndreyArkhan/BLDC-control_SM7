	I. Getting Started:

	1. Подключите стенд к компьютеру и источнику питания (инструкция по подключению находится в [Error 404])
	2. Запустите программу
	3. В окне "Стандартные настройки" нажмите Connect
	4. Выберите пункт раздела Режим
	5. Введите уставку в разделе Задание положения/скорости
	6. Нажмите Send

	II. Описание пользовательского интерфейса:

	1. Вкладка "Стандартные настройки" содержит функционал для включения стандартных режимов работы привода. Порядок его использования описан в
разделе I.

	2. Вкладка "Расширенные настройки" содержит расширенный функционал управления приводом. 
		а) Раздел "ПИД-регулятор" позволяет менять коэффициенты Kp / Ki / Kd (НЕ ИЗМЕНЯТЬ БЕЗ РАСЧЕТНЫХ ОБОСНОВАНИЙ!))
		б) Раздел "Плавный пуск/тормоз" позволяет плавно изменять скорость/положение. Для запуска: включите нужный Режим
("Стандартные настройки"); установите Время работы, с (время работы на полной скорости); нажмите Вкл; нажмите Send ("Стандартные настройки"). Для отключения нажмите Откл.
		в) Раздел "Закон изменения скорости от времени" позволяет задавать схему движения функциями пользователя в задаваемых диапазонах времени.
 По истечению времени процесс повторяется. Предупреждающие окна убраны, чтобы не раздражали, и закомментированы. Неправильно заданные и незаполненные функции
игнорируются. Функции должны распознаваться парсером, начало (t0) должно быть раньше конца (t1). Приоритет имеют Доп. Функции в порядке следования, при выходе
из диапазона управление передается Основной функции. Для запуска: включите нужный Режим ("Стандартные настройки"); заполните Основную и Доп. функции 
(для t1=0 - бесконечность, для t1x / t2x = 0 - ошибка); нажмите Вкл; нажмите Send ("Стандартные настройки"). Для отключения нажмите Откл.
	В функции аргумент - время t0 (не t!) значение - положение / скорость. Распознаваемые токены: числа, унарные операторы -х, +х, sin(), cos(), 
abs(), переменные t0, бинарные операторы x+y, x-y, x*y, x/y, x^y и т.д.
	Пример: 5+t0*(45+sin(t0/30-1/2)^3) 
	
	3. Вкладка "График" содержит средства для отображения и сохранения графика положения ротора привода. Названия элементов "Построение графика", "Очистить график", 
"Запись в файл", "Сохранить" соответствуют их назначению.
